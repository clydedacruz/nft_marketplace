# NFT marketplace contract 
1. Create Sale 
This should allow a user to place an NFT in escrow and and list a sale 
- NFT identifier 
- sale end time 
- minimum bid price (ether)

2. Bid for NFT 
- sale identifier 
- bid price in ether 

3. Claim item 
- sale ID 


## Pre-requisites
- Node.js
- Ganache 
- Truffle 

## Compiling contracts 
```
cd <>
```
